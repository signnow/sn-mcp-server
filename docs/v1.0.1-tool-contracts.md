## 1. `list_all_templates`

**Description:** Get simplified list of all templates and template groups with basic information *(+ tool fallback suffix)*

**Annotations:** `readOnlyHint=True`, `idempotentHint=True`

### Input

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `limit` | `int` (1–100) | No | `50` | Maximum number of items to return (1-100, default 50) |
| `offset` | `int` (≥0) | No | `0` | Number of items to skip for pagination (default 0) |

### Output → `TemplateSummaryList`

| Field | Type | Description |
|-------|------|-------------|
| `templates` | `list[TemplateSummary]` | Список шаблонов (см. ниже) |
| `total_count` | `int` | Total number of templates across all pages |
| `offset` | `int` | Number of items skipped |
| `limit` | `int` | Maximum number of items in this page |
| `has_more` | `bool` | Whether more items exist beyond this page |

#### `TemplateSummary`

| Field | Type | Description |
|-------|------|-------------|
| `id` | `str` | Template group ID |
| `name` | `str` | Template group name |
| `entity_type` | `str` | Type of entity: `'template'` or `'template_group'` |
| `folder_id` | `str \| None` | Folder ID if stored in folder |
| `last_updated` | `int` | Unix timestamp of last update |
| `is_prepared` | `bool` | Whether the group is ready for sending |
| `roles` | `list[str]` | All unique roles from all templates in the group |

## 2. `list_documents`

**Description:** Get simplified list of documents and document groups with basic information. Returns both documents and document groups in a unified format. Use this tool to fetch lists of documents by status, e.g. documents waiting for your signature (waiting-for-me) or expired documents (expired_filter=expired). *(+ tool fallback suffix)*

**Annotations:** `readOnlyHint=True`, `idempotentHint=True`

### Input

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `filter` | `Literal["signed", "pending", "waiting-for-me", "waiting-for-others", "unsent"] \| None` | No | `None` | Filter by document group status (optional). Available values: signed, pending, waiting-for-me, waiting-for-others, unsent. |
| `sortby` | `Literal["updated", "created", "document-name"] \| None` | No | `None` | Sort by created date, updated date, or document name (optional). Available values: updated, created, document-name. |
| `order` | `Literal["asc", "desc"] \| None` | No | `None` | Order of sorting (optional, can be used only with sortby). Available values: asc, desc. |
| `folder_id` | `str \| None` | No | `None` | Filter by folder ID (optional) |
| `expired_filter` | `Literal["all", "expired", "not-expired"]` | No | `"all"` | Filter by invite expiredness (optional, default: all). Available values: all, expired, not-expired. |
| `limit` | `int` (1–100) | No | `50` | Maximum number of items to return (1-100, default 50) |
| `offset` | `int` (≥0) | No | `0` | Number of items to skip for pagination (default 0) |

### Output → `SimplifiedDocumentGroupsResponse`

| Field | Type | Description |
|-------|------|-------------|
| `document_groups` | `list[SimplifiedDocumentGroup]` | Список документов/групп (см. ниже) |
| `document_group_total_count` | `int` | Total number of document groups across all pages |
| `offset` | `int` | Number of items skipped |
| `limit` | `int` | Maximum number of items in this page |
| `has_more` | `bool` | Whether more items exist beyond this page |

#### `SimplifiedDocumentGroup`

| Field | Type | Description |
|-------|------|-------------|
| `last_updated` | `int` | Unix timestamp of the last update |
| `id` | `str` | Document group or document ID |
| `name` | `str` | Name of the document group or document |
| `entity_type` | `str` | Type of entity: `'document'` or `'document_group'` |
| `invite` | `SimplifiedInvite \| None` | Unified invite info (см. [SimplifiedInvite](#simplifiedinvite)) |
| `documents` | `list[SimplifiedDocumentGroupDocument]` | List of documents in this group |

#### `SimplifiedDocumentGroupDocument`

| Field | Type | Description |
|-------|------|-------------|
| `id` | `str` | Document ID |
| `name` | `str` | Document name |
| `roles` | `list[str]` | Roles defined for this document |

## 3. `send_invite`

**Description:** Send invite to sign a document or document group. This tool is ONLY for documents and document groups. If you have template or template_group, use the alternative tool: `send_invite_from_template`

**Annotations:** `readOnlyHint=False`, `destructiveHint=False`, `idempotentHint=False`

### Input

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `entity_id` | `str` | **Yes** | — | ID of the document or document group |
| `orders` | `list[InviteOrder] \| str \| None` | No | `None` | List of orders with recipients (can be a list or JSON string) |
| `entity_type` | `Literal["document", "document_group"] \| None` | No | `None` | Type of entity: 'document' or 'document_group' (optional). If you're passing it, make sure you know what type you have. If it's not found, try using a different type. |

#### `InviteOrder`

| Field | Type | Required | Default | Description |

| `order` | `int` | **Yes** | — | Order number for this step |
| `recipients` | `list[InviteRecipient]` | **Yes** | — | List of recipients for this order |

#### `InviteRecipient`

| Field | Type | Required | Default | Description |
|-------|------|----------|---------|-------------|

| `role` | `str` | **Yes** | — | Recipient's role name in the document |

| `subject` | `str \| None` | No | `None` | Custom email subject for the recipient |

| `decline_redirect_uri` | `str \| None` | No | `None` | URL that opens after decline |

## MCP Resources (v1.0.1)

Кроме тулзов, в v1.0.1 были зарегистрированы MCP Resources (resource endpoints) — те же данные, но через `resources/read` для клиентов с поддержкой ресурсов:

| Resource URI | Backing tool |
|-------------|-------------|
| `signnow://templates{?limit,offset}` | `list_all_templates` |
| `signnow://documents/{?filter,sortby,order,folder_id,expired_filter,limit,offset}` | `list_documents` |
| `signnow://invite-status/{entity_id}{?entity_type}` | `get_invite_status` |
| `signnow://document-download-link/{entity_id}{?entity_type}` | `get_document_download_link` |
| `signnow://signing-link/{entity_id}{?entity_type}` | `get_signing_link` |
| `signnow://document/{entity_id}{?entity_type}` | `get_document` |

Параметры и возвращаемые модели у ресурсов идентичны соответствующим тулзам.
